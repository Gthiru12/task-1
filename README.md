# Linux File Permissions Task

This repository demonstrates hands-on Linux commands related to:

- File permissions (chmod)
- Ownership (chown)
- Root vs normal user behavior
- Directory permissions
- Practical troubleshooting

## Key Commands Used
- chmod
- chown
- ls -l
- stat
- su
- df
- free
- top

## Learning Outcome
Understood how Linux permission model works and why root can bypass file permissions.
